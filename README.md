# Development

## Installation
HyPhy Vision uses [yarn](https://yarnpkg.com/lang/en/docs/install/) for package management.

```
git clone https://github.com/veg/hyphy-vision
cd hyphy-vision
yarn
```

## Dev Server

Start the development server:

```
yarn start
```
